# Bsoyka Extras

## How do I install these formulae?

`brew install bsoyka/extras/<formula>`

Or `brew tap bsoyka/extras` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "bsoyka/extras"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
